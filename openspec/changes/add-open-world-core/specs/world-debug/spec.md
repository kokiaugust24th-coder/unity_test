# world-debug Spec Delta

## ADDED Requirements

### Requirement: エディタグリッド可視化

エディタは Scene ビューにワールドグリッドとセル状態(色分け: Unloaded/Loading/Loaded/Activated/AlwaysLoaded)をオーバーレイ表示できなければならない (SHALL)。セルを選択するとセル座標・オブジェクト数・HLOD 有無・所属レイヤーが確認できなければならない (SHALL)。

#### Scenario: プレイ中のセル状態表示
- **WHEN** プレイモード中に Scene ビューのオーバーレイを有効にする
- **THEN** 各セルが現在の状態に応じた色で表示される

### Requirement: ランタイム統計 HUD

システムはランタイムに表示可能な統計 HUD を提供しなければならない (SHALL)。HUD は最低限、状態別セル数、ロードキュー長、in-flight 数、直近フレームのインスタンス化予算消費、Addressables ハンドル数とバンドルメモリ概算を表示しなければならない (SHALL)。HUD は開発ビルドでのみ有効化され、リリースビルドのコードから除外可能でなければならない (SHALL)。

#### Scenario: 統計の表示
- **WHEN** 開発ビルドで HUD を有効にする
- **THEN** ストリーミング統計がリアルタイムに更新表示される

### Requirement: ストリーミングソースギズモ

エディタはストリーミングソースごとにアクティベート半径・ロード半径・HLOD 半径・ヒステリシス幅のリングをギズモ表示できなければならない (SHALL)。

#### Scenario: 半径リングの表示
- **WHEN** ストリーミングソースを選択する
- **THEN** 各半径のリングが Scene ビューに描画される

### Requirement: デバッグコマンド

システムは次のデバッグ操作を API およびデバッグ UI から実行できなければならない (SHALL): 全セル強制ロード、指定セルのロード/アンロード、データレイヤーの切替、ストリーミング一時停止/再開。

#### Scenario: 全ロード
- **WHEN** 全セル強制ロードを実行する
- **THEN** 全セルが距離判定を無視して `Activated` になる

#### Scenario: ストリーミング一時停止
- **WHEN** ストリーミングを一時停止してソースを移動する
- **THEN** セル状態は変化せず、再開後に現在位置基準で再評価される
