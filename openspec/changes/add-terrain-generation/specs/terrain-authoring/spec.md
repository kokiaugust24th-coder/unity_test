# terrain-authoring Spec Delta

## ADDED Requirements

### Requirement: 生成ウィンドウ

エディタは生成ウィンドウ (Tools > OpenWorld > Terrain Generator) を提供しなければならない (SHALL)。ウィンドウから設定アセットの選択、プレビュー生成、フル生成、進捗表示、キャンセルが行えなければならない (SHALL)。

#### Scenario: 生成の実行と進捗
- **WHEN** フル生成を実行する
- **THEN** ステージごとの進捗が表示され、途中キャンセルできる

### Requirement: 手動差分レイヤー

システムは Unity Terrain 上での手動ハイトマップ編集を「生成結果との差分」として記録し、再生成後に自動で再適用しなければならない (SHALL)。差分適用により安息角を大きく超える急勾配が生じた場合は警告しなければならない (SHALL)。

#### Scenario: 手修正が再生成で消えない
- **WHEN** 生成後に丘を手動でならし、ノイズパラメータを変更して再生成する
- **THEN** 新しい地形の上に手動修正が再適用されている

#### Scenario: 不整合の警告
- **WHEN** 再生成で差分適用箇所が崖になった
- **THEN** 該当領域が警告として報告される

### Requirement: ロックマスク

システムはペイント可能なロックマスクを提供しなければならない (SHALL)。ロック領域の高さ・スプラット・散布は再生成で一切変更されてはならない (MUST)。

#### Scenario: POI 周辺の完全固定
- **WHEN** 拠点周辺をロックして再生成する
- **THEN** ロック領域は高さ・テクスチャ・散布とも変化しない

### Requirement: 部分再生成

システムは矩形または選択セル範囲を指定した部分再生成を提供しなければならない (SHALL)。部分再生成の境界は既存地形と滑らかに接続されなければならない (SHALL)。

#### Scenario: 領域のみ再生成
- **WHEN** 1 セル分の範囲を指定して再生成する
- **THEN** 範囲内のみ更新され、境界に段差が生じない
