# terrain-generation Spec Delta

## ADDED Requirements

### Requirement: 決定論的な高さマップ合成

システムは `TerrainGenerationSettings` (シード、ノイズレイヤー、侵食パラメータ等) からワールド全体の高さマップを生成しなければならない (SHALL)。同一の設定・シード・Feature 入力からは、バイト一致の高さマップが再現されなければならない (MUST)。

#### Scenario: シード再現性
- **WHEN** 同一の設定アセットとシードで生成を 2 回実行する
- **THEN** 2 回の高さマップのハッシュが一致する

#### Scenario: シード変更
- **WHEN** シードのみ変更して生成する
- **THEN** 異なる地形が生成される

### Requirement: ノイズスタック合成

高さ合成は複数のノイズレイヤー (fBm / Ridged / Voronoi / Terrace) を順に合成できなければならない (SHALL)。各レイヤーはブレンドモード (加算・乗算・Min/Max)、振幅、周波数、適用マスクを持たなければならない (SHALL)。

#### Scenario: レイヤー合成
- **WHEN** ベース fBm に Ridged レイヤーを山岳マスク付きで加算合成する
- **THEN** マスク領域のみ尾根状の起伏が追加される

### Requirement: 大規模構造の合成

システムは台地・崖・大陸形状などの大規模構造をマスクベースで合成できなければならない (SHALL)。台地エッジには崖 (急峻な遷移帯) が形成されなければならない (SHALL)。

#### Scenario: 台地と崖
- **WHEN** 台地マスクを指定して生成する
- **THEN** マスク領域が高台になり、境界に崖の遷移帯が形成される

### Requirement: 侵食シミュレーション

システムは水力侵食 (ドロップレット方式) と熱侵食を高さマップへ適用できなければならない (SHALL)。侵食は谷・堆積・法面の緩和を生み、フロー (水流集積) と堆積のフィールドを出力しなければならない (SHALL)。侵食は並列実行でも決定論的でなければならない (MUST)。

#### Scenario: 侵食による谷の形成
- **WHEN** 侵食を有効にして生成する
- **THEN** 斜面に水流方向へ連続した谷が形成され、フローフィールドが出力される

#### Scenario: 侵食の決定論
- **WHEN** 同一シードで侵食を含む生成を 2 回実行する
- **THEN** 結果のハッシュが一致する

### Requirement: フィールド可視化デバッガ

エディタは高さ・フロー・堆積・傾斜・バイオームの各フィールドをヒートマップとして可視化できなければならない (SHALL)。これはチューニング反復の基盤であり、侵食実装より先に提供されなければならない (SHALL)。

#### Scenario: フローの可視化
- **WHEN** 生成後にフローフィールドの可視化を選択する
- **THEN** 水流の集積が Scene 上またはプレビュー画像で確認できる

### Requirement: 外部ハイトマップのインポート (補助入力)

システムは外部ツールが出力したハイトマップとマスク (RAW16 / EXR / PNG) を高さフィールドとしてインポートできなければならない (SHALL)。インポート時にワールドサイズ・最大標高のマッピングを指定でき、解像度不一致はバイリニア補間でリサンプルされなければならない (SHALL)。

#### Scenario: 外部出力の取り込み
- **WHEN** 4097px の RAW16 ハイトマップを 4km / 最大標高 800m としてインポートする
- **THEN** 高さフィールドが生成され、以降のステージ (Feature・バイオーム・タイル化) の入力になる

### Requirement: バイオーム分類

システムは高度・傾斜・湿度 (ノイズ + フロー由来) からバイオーム ID フィールドを生成しなければならない (SHALL)。バイオームは設定アセットで任意数定義できなければならない (SHALL)。

#### Scenario: 高度によるバイオーム遷移
- **WHEN** 低地に草原、高地に岩場バイオームを定義して分類する
- **THEN** 標高に応じてバイオーム ID が遷移する

### Requirement: スプラットマップ生成

システムはバイオーム定義の TerrainLayer ルール (傾斜による岩肌露出等) に従い、Terrain 用スプラットマップを生成しなければならない (SHALL)。

#### Scenario: 急傾斜の岩肌
- **WHEN** 傾斜 40° 以上を岩レイヤーとするルールで生成する
- **THEN** 崖面に岩テクスチャが割り当てられる

### Requirement: プレビューモード

システムは低解像度 (既定 1/4) で全ステージを実行するプレビューモードを提供しなければならない (SHALL)。プレビューはフル解像度と同じシード系列を使わなければならない (SHALL)。

#### Scenario: 高速プレビュー
- **WHEN** プレビューモードで生成する
- **THEN** フル解像度より大幅に短い時間で概形が確認できる
